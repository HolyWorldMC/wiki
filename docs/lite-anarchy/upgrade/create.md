# ♻️ Улучшение предметов

Улучшения — особое меню, позволяющее превращать предметы в предметы буквально уровнем выше за боевые фрагменты.

## Как открыть меню улучшений

Меню улучшений предметов доступно по команде `/create`.

## Как работает улучшение

<div data-with-frame="true"><figure><img src="../.gitbook/assets/create-upgrade.png" alt=""><figcaption><p>Главное меню Улучшения предметов /create</p></figcaption></figure></div>

Имея достаточно много Боевых фрагментов, вы можете улучшить почти любой предмет, который у вас есть. В открытом меню улучшений `/create` положите желаемый предмет для улучшения в специальный слот. Далее, если у вас достаточно боевых фрагментов, подтвердите свою попытку на улучшение.

{% hint style="info" %}
Чтобы увеличить шансы на улучшение предмета без его потери, вы можете добавить до пяти такких же предметов.
{% endhint %}

## Применение сломанной брони

<div data-with-frame="true"><figure><img src="../.gitbook/assets/create-example.png" alt=""><figcaption><p>Пример того, как работает улучшение /create</p></figcaption></figure></div>

С помощью механики улучшений `/create` вы можете избавиться от сломанных предметов, на которые нельзя наложить починку, и большого количества топовой брони в сундуках. Использование механики улучшений позволит вам как очистить сундуки, так и одновременно стать еще сильнее.

{% hint style="info" %}
Текущая прочность нового предмета возрастет в зависимости от материала, из которого он будет изготовлен. Например, кожа будет прочнее, чем железо.
{% endhint %}
