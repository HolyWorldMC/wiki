# 📊 Биржа

Биржа — это торговая площадка для обмена монеток на коины, которые приобретаются за реальную валюту на официальном сайте сервера. Система позволяет игрокам торговать между собой игровой валютой.

## Как открыть биржу

<div data-with-frame="true"><figure><img src="../.gitbook/assets/exchange-menu.png" alt=""><figcaption><p>Главное меню Биржи</p></figcaption></figure></div>

Меню биржи доступно по команде `/exchange`.

## Как обменять коины на монетки

### Экспресс

Обменять коины на монетки можно быстро и просто. Для этого нужно ввести команду `/exchange buy <количество монеток>`. Укажите, сколько монеток вы хотите получить, и система автоматически найдет выгодный обмен и обменяет ваши коины на нужное количество монет.

{% hint style="info" %}
Пример использования команды /exchange buy

* /exchange buy 100000 — вы получите 100.000 монеток за свои коины



Дабы упростить указание количества монеток, вы можете использовать следующий способ:

* Указание цены при помощи букв: 1k — 1000 монеток, 1kk — 1 000 000 монеток, 1m — 1 000 000 монеток. Пример: /exchange buy 5kk — вы получите 5 000 000 монеток;


{% endhint %}

### Маркет

<div data-with-frame="true"><figure><img src="../.gitbook/assets/exchange-coins-to-money.png" alt=""><figcaption><p>Одна из кнопок, чтобы обменять свои коины на монетки</p></figcaption></figure></div>

Обменять коины на монетки можно в главном меню биржи `/exchange`. Выберите желаемую заявку и нажмите по иконке заявки. Вы можете выкупить как все сразу монетки, так и выкупить только часть монеток.

{% hint style="info" %}
В главном меню биржи заявки сортируются по выгодности. Сначала показываются те, где за один коин предлагается больше всего монеток.
{% endhint %}

## Как обменять монетки на коины

<div data-with-frame="true"><figure><img src="../.gitbook/assets/exchange-money-to-coins (1).png" alt=""><figcaption><p>Кнопка, чтобы подвердить создание заявки на обмен монеток на коины</p></figcaption></figure></div>

Обменять монетки на коины можно в главном меню биржи `/exchange`. Для этого нужно создать заявку на продажу монет, нажав на специальную иконку в меню. У вас откроется меню с выбором, сколько вы хотите получить коинов.
