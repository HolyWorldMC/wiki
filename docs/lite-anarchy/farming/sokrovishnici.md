# 💰 Сокровищницы

Сокровищницы — это особые структуры, где блок региона охраняет сундуки с ценным лутом. Для получения доступа к содержимому необходимо уничтожить блок привата при помощи динамита.

## Получить доступ к сундукам

Каждая сокровищница защищена блоком привата, который необходимо взорвать при помощи динамита.

{% hint style="info" %}
Блок привата может быть в виде блока «Якоря возрождения», который уничтожается при помощи только С4 или «Разрывной волны». А также или в виде блока «Магнетит», который уничтожается любым динамитом.
{% endhint %}

## Типы сокровищниц

Существует **8+ различных видов** сокровищниц, отличающихся уровнем защиты и ценностью содержимого. Ценность содержимого сокровищницы прямо пропорциональна сложности её вскрытия. Чем тяжелее разрушается блок привата, тем ценнее содержимое сокровищницы.

{% hint style="info" %}
Сокровищницы генерируются случайным образом по всему миру. Их обнаружение требует активного исследования территории или использования специальных предметов, таких как особый компас.
{% endhint %}
